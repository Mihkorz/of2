Methods

Seven functional groups were considered in the differential gene expression analysis: ES, EPL, ASC, ANC, AEC, ABC and CCL. In all following parts of analysis EPL group was used as a reference and other groups were compared to it. For the gene-level analysis, we performed different statistical approaches among which a negative binomial distribution-based approach appeared to be the most powerful and statistically grounded. This distribution can be considered as an extension of Poisson distribution and can model both shot noise generated by random subsampling and sequencing and biological variation between samples. This was followed by Benjamini-Hochberg FDR adjustment [1]. For the pathway level analysis each preprocessed gene expression group was independently compared to EPL reference using an algorithm called OncoFinder [2]. Using preprocessed gene expression data as an input allows for cross-platform dataset comparison with low error rate and provides the ability to obtain functional features of intracellular regulation using mathematical estimations. Positive and negative PAS values indicate pathway up- and down-regulation, respectively. In total, we performed the analysis of 1856 signaling and 319 metabolic pathways.
